# Small Web Projects (HTML, CSS & JavaScript)

This repository contains a collection of **small web projects** built using **HTML, CSS, and JavaScript**.  
These projects are mainly created for **JavaScript practice and experimentation**.

They are **basic, non-polished projects**, not intended as major or portfolio-level work.

## What's Inside

- Multiple small projects, each in its own folder
- Focus on core JavaScript concepts
- Simple UI built with HTML and CSS
- Examples may include games, utilities, or mini-features

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Purpose of This Repository

- Practice JavaScript fundamentals
- Experiment with logic and DOM manipulation
- Learn by building small things consistently

## Notes

- Projects may vary in quality and completeness
- Code is written for learning purposes
- No frameworks or libraries used

## How to Use

1. Open any project folder
2. Run the project by opening `index.html` in a browser
